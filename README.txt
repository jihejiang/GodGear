 /give @s godgear:god_sword
 /give @s godgear:god_spear
 /give @s godgear:god_mace
 /give @s godgear:god_helmet
 /give @s godgear:god_chestplate
 /give @s godgear:god_leggings
 /give @s godgear:god_boots

Leggings toggle:
